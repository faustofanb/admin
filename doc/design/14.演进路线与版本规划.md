## 14.1 版本规划

### 14.1.1 v1.0 MVP（最小可行性产品）
- **目标**：跑通核心流程，支持单租户/多租户基础运营。
- **功能**：
  - 用户/角色/权限管理（RBAC）。
  - 基础多租户隔离。
  - 资源上传/下载（本地存储或简单 OSS）。
  - 基础日志与监控。

### 14.1.2 v1.5 增强版
- **目标**：提升性能与稳定性，支持复杂业务。
- **功能**：
  - 引入 **RustFS** 替换本地存储。
  - 引入 **PowerJob** 替换 Spring Scheduled。
  - 引入 **EasyExcel** 异步导入导出。
  - 完善 **Sentinel** 限流熔断。

### 14.1.3 v2.0 分布式版
- **目标**：支持大规模并发，模块可独立部署。
- **功能**：
  - **微服务拆分**：将 `module-resource`, `module-report` 拆分为独立进程。
  - **Seata**：全面启用分布式事务。
  - **SkyWalking/OTel**：全链路追踪可视化。
  - **K8s**：容器化编排与自动扩缩容。

## 14.2 技术演进路线

### 14.2.1 架构演进
- **阶段一（当前）**：单体模块化（Modular Monolith）。部署简单，调用高效。
- **阶段二**：核心单体 + 边缘微服务。将资源、报表等独立。
- **阶段三**：Service Mesh。业务逻辑与基础设施完全解耦。

### 14.2.2 存储演进
- **DB**：PostgreSQL 主从 -> TiDB / OceanBase（分布式数据库）。
- **Cache**：Redis 主从 -> Redis Cluster。
- **File**：RustFS -> Ceph / MinIO 集群。
